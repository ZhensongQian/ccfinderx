ccfinderx
=========

To compile the project on the development PC execute:
1) autoreconf -i
2) cd build
3) ../configure
4) make


Dependencies:
autoconf, automake, build-essential, 
libboost-all-dev,
openjdk-7-dev, ant, 
libicu4j-java, libswt-gtk-java, libtrove-java, libswt-cairo-gtk-3-jni

To run the project execute:
cd build
gemx.sh
